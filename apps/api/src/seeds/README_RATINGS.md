# Сид для системы рейтингов

## Описание

Сид `seed-ratings.ts` создает тестовые данные для системы рейтингов:
- Результаты событий (EventResults)
- Статистику игроков (PlayerStats)
- Статистику команд (TeamStats)
- Рейтинги игроков и команд

## Что делает сид

1. **Находит завершенные события** (или помечает некоторые как завершенные)
2. **Создает результаты событий** для участников:
   - Для командных событий: создает результаты для команд
   - Для индивидуальных событий: создает результаты для игроков
   - Генерирует случайную статистику (убийства, смерти, точность)
   - Рассчитывает очки по формуле
   - Помечает результаты как подтвержденные (CONFIRMED)

3. **Обновляет статистику игроков**:
   - Подсчитывает игры, победы, поражения, ничьи
   - Рассчитывает общие очки, средние очки, win rate
   - Рассчитывает K/D ratio и точность

4. **Обновляет статистику команд**:
   - Подсчитывает игры и победы
   - Рассчитывает общие очки и win rate

5. **Пересчитывает рейтинги**:
   - Сортирует игроков/команды по totalPoints
   - Присваивает ранги (1, 2, 3, ...)

## Запуск

### Отдельно
```bash
npm run seed:ratings --workspace=api
```

### В составе полной настройки БД
```bash
npm run setup:db --workspace=api
```

## Требования

Перед запуском сида должны быть:
- ✅ Созданы события (через `seed:events`)
- ✅ Созданы команды (через `seed:teams`)
- ✅ Пользователи зарегистрированы на события

## Генерируемые данные

### Статистика для результатов:
- **Убийства**: 5-25 для команд, 3-18 для игроков
- **Смерти**: 2-12 для команд, 1-9 для игроков
- **Точность**: 60-90% для команд, 65-90% для игроков

### Места:
- Первое место (FIRST) - 1 команда/игрок
- Второе место (SECOND) - 1 команда/игрок
- Третье место (THIRD) - 1 команда/игрок
- Участие (PARTICIPATED) - остальные

## Формула расчета очков

```
Очки = Базовые_очки_за_место + (Убийства × 2) + (Точность × 0.5)

Где:
- FIRST = 100 очков
- SECOND = 75 очков
- THIRD = 50 очков
- PARTICIPATED = 25 очков
```

## Примечания

- Сид очищает существующие результаты перед созданием новых
- Результаты создаются только для событий с одобренными регистрациями
- Статистика рассчитывается только для подтвержденных результатов
- Рейтинги пересчитываются автоматически
